# Amazon Sales Data Analysis

## Project Overview
This project analyzes Amazon sales data to uncover insights related to revenue, profitability, customer behavior, and discount strategies.

---

## Objectives
- Understand overall business performance
- Identify high-performing and underperforming categories and regions
- Analyze the impact of discounts on sales and profit
- Provide actionable business insights

---

## Tools Used
- SQL (Data Analysis)
- Power BI (Dashboard & Visualization)
- Excel (Data Cleaning)

---

## Key Insights
- Higher discounts increased sales volume but reduced profitability
- Some categories generated high revenue but low profit margins
- Certain regions underperformed in both revenue and profit
- Customer payment preferences showed clear usage patterns
